# 各服务模块端口约定

本地/ dev 环境同时启动多模块时，端口需互不冲突。以下为当前约定（仅列主配置中的 `server.port`，不含 target 生成文件）。

| 端口   | 模块           | 说明 |
|--------|----------------|------|
| 6001   | ucenter-api    | 用户中心 API |
| 6002   | exchange       | 交易撮合引擎（原 6005，为避免与 clearing 冲突改为 6002） |
| 6003   | exchange-api   | 交易 API |
| 6004   | market         | 行情 / Market |
| 6005   | clearing       | 清算服务 |
| 6006   | settlement     | 结算服务 |
| 6007   | fund           | 资金服务 |
| 6009   | wallet-core    | 钱包核心 |
| 6013   | exchange-relay | 订单中继 |
| 7003   | wallet/eth     | 以太坊钱包 |
| 7004   | wallet/erc-eusdt | ERC-EUSDT 钱包 |
| 7421   | cloud          | Eureka 注册中心 |
| 7421   | test           | 测试模块 |
| 8001   | core           | 仅测试资源（core 为 lib，一般不单独起服务） |

**注意**：修改任一模块端口时，请同步更新本文档，并确保不与上表其它端口重复。
